plot_mobo_points_in_obj_space(obs_values: trieste.types.TensorType, num_init: int | None = None, mask_fail: trieste.types.TensorType | None = None, figsize: tuple[float, float] | None = (8, 6), xlabel: str = 'Obj 1', ylabel: str = 'Obj 2', zlabel: str = 'Obj 3', title: str | None = None, m_init: str = 'x', m_add: str = 'o', c_pass: str = 'tab:green', c_fail: str = 'tab:red', c_pareto: str = 'tab:purple', only_plot_pareto: bool = False) → tuple[matplotlib.figure.Figure, matplotlib.axes.Axes][source]#

Adds scatter points in objective space, used for multi-objective optimization (2 or 3 objectives only). Markers and colors are chosen according to BO factors.

Parameters:

  • obs_values – TF Tensor or numpy array of objective values, shape (N, 2) or (N, 3).

  • num_init – initial number of BO points

  • mask_fail – Bool vector, True if the corresponding observation violates the constraint(s)

  • figsize – Size of the figure.

  • xlabel – Label of the X axis.

  • ylabel – Label of the Y axis.

  • zlabel – Label of the Z axis (in 3d case).

  • title – Title of the plot.

  • m_init – Marker for initial points.

  • m_add – Marker for the points observed during the BO loop.

  • c_pass – color for the regular BO points

  • c_fail – color for the failed BO points

  • c_pareto – color for the Pareto front points

  • only_plot_pareto – if set to True, only plot the pareto points. Default is False.

Returns:

figure and axes
